Semantic HTML

Semantic HTML introduces meaning to the web page rather than just presentation. It makes web content more accessible and SEO-friendly.

Why Semantic HTML?

  • Better accessibility for screen readers
  • Improved SEO
  • Easier to maintain and understand
  • Future-proof your code

Common Semantic Elements

<header>
    <h1>Website Title</h1>
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
        </ul>
    </nav>
</header>

<main>
    <article>
        <h2>Article Title</h2>
        <p>Article content...</p>
    </article>
    
    <aside>
        <h3>Related Content</h3>
        <p>Sidebar content...</p>
    </aside>
</main>

<footer>
    <p>© 2023 My Website</p>
</footer>
