DOM Manipulation

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document and allows JavaScript to manipulate its content and style.

Selecting Elements

// Select by ID
const header = document.getElementById('header');

// Select by class name
const items = document.getElementsByClassName('item');

// Select by tag name
const paragraphs = document.getElementsByTagName('p');

// Modern selectors (CSS-like)
const element = document.querySelector('.container');
const allElements = document.querySelectorAll('.item');

Modifying Elements

// Change content
element.textContent = 'New text content';
element.innerHTML = '<strong>HTML content</strong>';

// Modify attributes
element.setAttribute('class', 'new-class');
element.classList.add('active');
element.classList.remove('old-class');

// Change styles
element.style.color = 'blue';
element.style.fontSize = '20px';
